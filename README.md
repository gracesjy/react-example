## Installation
Ubuntu
```
   curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash
   source ~/.bashrc
   nvm install --lts
```
Get Source
```
   mkdir GitHub
   cd GitHub
   git clone https://github.com/gracesjy/react-example.git
   cd ~/GitHub/react-example
   npm install --save-dev vite
```
Run
```
   npm run dev
```
